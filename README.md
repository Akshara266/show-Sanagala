### Python Snippet 8: Fibonacci Sequence
This snippet calculates the 8th Fibonacci number using recursion.
 def fibonacci(n):
     if n <= 0:
        return "Invalid input"
     elif n == 1:
        return 0
     elif n == 2:
        return 1
     else:
        return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(8))

**Output**: 13
